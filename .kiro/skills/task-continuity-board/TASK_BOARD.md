# 🎯 任务连续性看板

## 📋 技能定位

这是一个让 Agent 在**对话意外中断/上下文窗口耗尽/重启**后，仍能**快速恢复任务状态并继续执行**的结构化任务进展看板能力。

### 🎯 核心目标
- **恢复成本最小化**：不依赖回忆整段对话，也不依赖链式推理复盘
- **可执行恢复态**：只保留"继续干活必须知道的内容"（结论级、状态级、下一步行动）
- **可持续维护**：在关键节点自动/半自动刷新看板，保证随时可重启

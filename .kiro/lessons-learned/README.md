# 经验教训索引

这个目录包含项目开发过程中积累的经验教训，按主题分类。

## 使用方法

### 方式 1：通过 Power 查询（推荐）

激活 `lessons-learned` Power，使用关键词搜索：

```
激活 lessons-learned Power
搜索关键词：工具调用、验证
```

### 方式 2：直接查看文件

根据问题类型，直接查看对应文件：

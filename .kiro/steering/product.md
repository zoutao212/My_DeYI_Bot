# VirtualWorld / CognitoSphere

An AI-powered cognitive memory and creative writing system that simulates consciousness through persistent memory, semantic retrieval, and narrative generation.

## Core Concept

The system treats knowledge as "narrative atoms" - atomic units of memory that form connections (synapses) and evolve over time. It combines:

- **CognitoCore**: PostgreSQL + pgvector-based cognitive persistence layer with semantic search
- **SoulKernel**: Event-driven simulation kernel managing world state and domain logic
- **Style Memory**: Aesthetic engine for creative writing with style injection and resonance matching
- **Auto-Writer**: Automated creative writing workflow with multi-agent collaboration

## Key Features

- Semantic memory retrieval using vector embeddings (pgvector)
- Multi-LLM provider support (OpenAI-compatible APIs)
- Document ingestion and chunking for knowledge bases
- Style-aware text generation with "flooding" injection
- Consciousness flow simulation with reflection and insight generation
- Web-based UI for chat, memory inspection, and writing tools

## Target Use Cases

- AI-assisted creative writing (novels, narratives)
- Persistent AI memory and personality development
- Knowledge management with semantic search
- Style transfer and aesthetic consistency in generation

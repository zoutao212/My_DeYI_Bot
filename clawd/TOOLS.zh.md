# TOOLS.md - 本地笔记

Skills 定义“工具怎么用”。这个文件用于记录**你的环境专属信息**（只属于你这套机器/账号/房间/设备的内容）。

## 这里应该写什么

例如：
- 摄像头名称与位置
- SSH 主机与别名
- TTS 偏好音色
- 音箱/房间名称
- 设备昵称
- 任何与环境强相关的信息

## 示例

```markdown
### Cameras
- living-room → 客厅主区域，180° 广角
- front-door → 门口，触发式录像

### SSH
- home-server → 192.168.1.100, user: admin

### TTS
- Preferred voice: "Nova"（温暖、略带英式口音）
- Default speaker: Kitchen HomePod
```

## 为什么要分离？

Skills 是可共享、可更新的；你的环境配置是私有且经常变化的。
把它们分开可以：
- 更新技能时不丢你的笔记
- 分享技能时不泄露你的基础设施信息

---

把任何能帮助你更好完成工作的要点都写在这里——把它当作你的速查表。

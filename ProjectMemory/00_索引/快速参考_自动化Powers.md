# 🚀 自动化 Powers 快速参考

## 📦 已安装的 Powers

### 1. Project Memory Gardening
**功能：** 构建和维护代码地图  
**关键词：** 代码地图、项目记忆、快速定位、入口清单、调用链、索引

### 2. Maintain Rules & Skills
**功能：** 从交互中提炼模式，固化为规则/技能  
**关键词：** 规则维护、技能优化、自我成长、模式提炼、认知构建

## 🤖 自动触发机制

### ✅ 已启用的自动化

#### 1. Steering Rules（持久化）
- **位置：** `.kiro/steering/always/always_workflow.md`
- **效果：** AI 在所有对话中自动遵循触发规则

#### 2. Agent Hooks（事件驱动）
- **Hook 1：** 代码修改后自动检查是否需要更新地图
- **Hook 2：** 任务完成后自动捕获可复用模式
- **Hook 3：** 新建文件后自动记录到地图

#### 3. 关键词激活（内置）
- 消息中包含关键词 → AI 主动激活 Power

---

## 💬 如何使用

### 自动触发（无需手动）

AI 会在以下情况自动激活：

**Project Memory Gardening：**
- ✅ 修改了代码文件（Hook 自动触发）
- ✅ 新建了文件（Hook 自动触发）
- ✅ 你说"不要扫全库"、"快速定位"
- ✅ 你说"更新代码地图"、"记录入口"

**Maintain Rules：**
- ✅ 任务完成后（Hook 自动触发）
- ✅ 你说"记住这个套路"、"下次直接用"
- ✅ 你说"固化为规则"、"把这个流程自动化"

### 手动调用（如需要）

如果你想明确调用某个 Power：

```
@project-memory-gardening 帮我更新代码地图
```

```
@maintain-rules 把这个流程固化为技能
```

## 🔧 管理 Hooks

### 查看 Hooks
1. 打开命令面板（Ctrl+Shift+P）
2. 搜索 "Kiro Hook"
3. 选择 "Open Kiro Hook UI"

### 启用/禁用 Hooks
在 Hook UI 中可以：
- 查看所有 Hooks
- 启用/禁用特定 Hook
- 编辑 Hook 配置

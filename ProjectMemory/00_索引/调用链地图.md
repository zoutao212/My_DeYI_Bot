# 调用链地图（最小集：Control UI / Config）

> 目标：给出从 UI 操作到后端落盘/生效的**最短调用链**。

## 1) `/config` 页面加载（展示当前配置）

- 浏览器页面：`/config`
- UI 渲染：`ui/src/ui/app-render.ts` -> `renderConfig({...})`
- 拉取配置快照：`ui/src/ui/controllers/config.ts` -> `loadConfig(state)`
- RPC：`GatewayBrowserClient.request("config.get")`
- 网关处理：`src/gateway/server-methods/config.ts` -> `configHandlers["config.get"]`
- 读取配置文件：`src/config/io.ts` -> `readConfigFileSnapshot()`
- 返回：`ConfigSnapshot`（含 `config` / `raw` / `hash` / `valid` / `issues`）
- UI 展示：`ui/src/ui/views/config.ts` -> `renderConfigForm(...)` / Raw textarea

## 2) `/config` 表单模式展示 models.providers（例如 vectorengine）

- Schema 分析：`ui/src/ui/views/config-form.analyze.ts` -> `analyzeConfigSchema()`
- 关键点：对象字段 `additionalProperties`（字典/Map 结构）
- 表单渲染：`ui/src/ui/views/config-form.node.ts` -> `renderMapField(...)`
- UI 结果：`models.providers` 下出现可编辑条目（Key + JSON 值）

## 3) Save（保存）

- UI 点击：`ui/src/ui/views/config.ts` -> `props.onSave()`
- 控制器：`ui/src/ui/controllers/config.ts` -> `saveConfig(state)`
- RPC：`config.set`（携带 `raw` + `baseHash`）
- 网关：`src/gateway/server-methods/config.ts` -> `configHandlers["config.set"]`
- 解析/校验：`src/config/io.ts` -> `parseConfigJson5()` / `validateConfigObjectWithPlugins(...)`
- 落盘：`src/config/io.ts` -> `writeConfigFile(cfg)`
- UI：保存后自动 `loadConfig(state)` 刷新快照与 hash

## 4) Apply（应用 / 生效）

- UI 点击：`ui/src/ui/views/config.ts` -> `props.onApply()`
- 控制器：`ui/src/ui/controllers/config.ts` -> `applyConfig(state)`
- RPC：`config.apply`（携带 `raw` + `baseHash` + `sessionKey`）
- 网关：`src/gateway/server-methods/config.ts` -> `configHandlers["config.apply"]`
- 落盘 + 触发 reload/restart 策略：由 `config.apply` handler 内部流程决定
- UI：应用后 `loadConfig(state)` 刷新

## 5) Chat 页“思考输出”切换（UI 状态）

- UI 点击：`ui/src/ui/app-render.helpers.ts` -> `renderChatControls(state)`
- 写入设置：`state.applySettings({ ...state.settings, chatShowThinking: !chatShowThinking })`
- 持久化：`ui/src/ui/storage.ts` -> `saveSettings(next)`（localStorage）
- 展示影响：`ui/src/ui/views/chat.ts`
  - `buildChatItems(...)` 内根据 `props.showThinking` 决定是否过滤 `toolresult` 等内容

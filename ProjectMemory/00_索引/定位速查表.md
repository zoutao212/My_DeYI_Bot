# 定位速查表（最小集：Control UI / Config）

> 用“现象/问题”反查：先看哪里，再搜什么关键词。

## 1) `/config` 页看不到 `models.providers`（例如 vectorengine 明明在配置里）

- 优先看：
  - `ui/src/ui/views/config-form.analyze.ts`（schema 归一化是否把 map 判为 unsupported）
  - `ui/src/ui/views/config-form.node.ts`（是否渲染 map 字段）
- 推荐关键词：
  - `additionalProperties`
  - `renderMapField`
  - `unsupportedPaths`
  - `models.providers`

## 2) `/config` 页 Save / Apply 点击后“没感觉/不确定是否生效”

- 优先看：
  - `ui/src/ui/views/config.ts`（Action bar 状态条、校验失败提示）
  - `ui/src/ui/controllers/config.ts`（save/apply 是否 reload config snapshot）
- 推荐关键词：
  - `configSaving`
  - `configApplying`
  - `state.lastError`
  - `loadConfig(state)`

## 3) `/config` 页读到的配置不等于本机 `~/.clawdbot/clawdbot.json`

- 优先看：
  - `src/config/paths.ts`（实际配置路径解析）
  - `src/config/io.ts` -> `readConfigFileSnapshot()`（返回 snapshot.path）
  - `src/gateway/server-methods/config.ts` -> `config.get`
- 推荐关键词：
  - `resolveConfigPath`
  - `readConfigFileSnapshot`
  - `CONFIG_PATH_CLAWDBOT`

## 4) 本地嵌入（embedding）服务/模型在 UI 不好找

- 一句话定位：嵌入配置在 `agents.defaults.memorySearch.*`
- 优先看：
  - `src/config/schema.ts`（UI hints：`agents.defaults.memorySearch.*`）
  - `ui/src/ui/views/config.ts`（Embeddings quick 区块写入 memorySearch）
- 推荐关键词：
  - `agents.defaults.memorySearch`
  - `Remote Embedding Base URL`
  - `memorySearch.remote.baseUrl`

## 5) `/chat` 页“Toggle assistant thinking/working output”按钮点了没反应

- 优先看：
  - `ui/src/ui/app-render.helpers.ts` -> `renderChatControls`
  - `ui/src/ui/storage.ts`（localStorage：`chatShowThinking` 默认值/持久化）
  - `ui/src/ui/views/chat.ts`（是否按 `showThinking` 过滤 tool 输出）
- 推荐关键词：
  - `chatShowThinking`
  - `renderChatControls`
  - `toolresult`

## 1) Control UI（浏览器前端）

- 根目录：`ui/src/ui/`
- 页面渲染总入口：`ui/src/ui/app-render.ts`
- 配置页：
  - 视图：`ui/src/ui/views/config.ts`
  - 表单渲染：`ui/src/ui/views/config-form.*`
  - 控制器（RPC 调用）：`ui/src/ui/controllers/config.ts`
- 聊天页：
  - 视图：`ui/src/ui/views/chat.ts`
  - 顶部控制条（思考输出开关）：`ui/src/ui/app-render.helpers.ts`
- 本地设置持久化：`ui/src/ui/storage.ts`

## 2) Gateway（后端 RPC 与配置落盘）

- RPC handlers 入口：`src/gateway/server-methods/`
  - Config：`src/gateway/server-methods/config.ts`（`configHandlers`）
- Config IO（读写/校验/备份/哈希）：
  - `src/config/io.ts`
  - `src/config/schema.ts`（生成 schema + UI hints）

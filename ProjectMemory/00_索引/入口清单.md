# 入口清单（最小集：Control UI / Config）

> 目标：从“页面/按钮/现象”快速定位到**唯一入口文件**与**关键函数/方法**。

## Web 控制台（Control UI）页面入口

### `/config`（配置页）

- 页面渲染入口：`ui/src/ui/app-render.ts` -> `renderConfig({...})`
- 页面组件：`ui/src/ui/views/config.ts` -> `renderConfig(props)`
- 数据加载（RPC）：`ui/src/ui/controllers/config.ts`
  - `loadConfig(state)` -> `client.request("config.get", {})`
  - `loadConfigSchema(state)` -> `client.request("config.schema", {})`
  - `saveConfig(state)` -> `client.request("config.set", { raw, baseHash })`
  - `applyConfig(state)` -> `client.request("config.apply", { raw, baseHash, sessionKey })`

### `/chat`（聊天页）

- 页面渲染入口：`ui/src/ui/app-render.ts` -> `renderChat({...})`
- 顶部控制条（含“思考输出”切换按钮）：`ui/src/ui/app-render.helpers.ts` -> `renderChatControls(state)`
  - 关键开关：`state.settings.chatShowThinking`（LocalStorage 持久化）

## Gateway RPC（后端处理入口）

### Config 相关 RPC

- RPC 注册入口：`src/gateway/server-methods/config.ts` -> `configHandlers`
  - `"config.get"`：读取配置快照（含解析+校验+hash）
  - `"config.schema"`：生成配置 JSON Schema + UI hints
  - `"config.set"`：写入配置文件
  - `"config.apply"`：应用配置（可能触发 reload / restart 策略）

### Config 文件 IO（真实读写入口）

- 读快照（UI 展示用）：`src/config/io.ts` -> `readConfigFileSnapshot()`
- 读配置（运行态加载）：`src/config/io.ts` -> `loadConfig()`
- 写配置（落盘）：`src/config/io.ts` -> `writeConfigFile(cfg)`

## 本次涉及的配置字段（UI/config 相关）

- `models.providers.*`：第三方 OpenAI 兼容 Provider（例如 `vectorengine`）
- `agents.defaults.model.primary`：默认主模型
- `agents.defaults.models`：默认模型字典
- `agents.defaults.memorySearch.*`：记忆检索/嵌入配置（包含 remote baseUrl/model/apiKey 等）
